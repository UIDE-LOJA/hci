Esta actividad práctica te permitirá aplicar principios de Gestalt en el rediseño de una interfaz, utilizando herramientas de IA para generar propuestas visuales. Te enfocarás en identificar problemas perceptuales, aplicar principios de organización visual, y justificar decisiones de diseño.

¿Qué realizarás?

Actividad Instrucciones Entregable
1) Selección de interfaz Elige una interfaz existente (web o móvil) con problemas de organización visual. Captura pantalla del estado actual ("Before"). Captura de pantalla original con anotaciones de problemas.
2) Análisis de problemas Identifica al menos 3 problemas relacionados con principios de Gestalt: proximidad, similitud, cierre, continuidad, figura-fondo. Lista de problemas con principio de Gestalt violado.
3) Propuesta de rediseño Usa herramientas de IA (ChatGPT, Midjourney, Figma AI, etc.) para generar propuestas de rediseño aplicando principios de Gestalt. Imagen "After" generada con IA.
4) Justificación técnica Explica cómo cada cambio aplica un principio de Gestalt específico y mejora la percepción visual. Tabla comparativa Before/After con justificaciones.
5) Presentación breve Prepara presentación de 2-3 minutos mostrando Before/After y explicando mejoras aplicadas. Slides o documento visual para presentación.

Principios de Gestalt a aplicar:

  • Proximidad: Elementos cercanos se perciben como relacionados
  • Similitud: Elementos similares se agrupan perceptualmente
  • Cierre: Tendencia a completar formas incompletas
  • Continuidad: Preferencia por líneas y patrones continuos
  • Figura-fondo: Separación entre objeto de enfoque y fondo
  • Conexión uniforme: Elementos conectados se perciben como grupo

Herramientas de IA sugeridas:

  • ChatGPT + DALL-E: Generación de mockups con prompts descriptivos
  • Midjourney: Creación de interfaces visuales
  • Figma AI: Asistencia en diseño de interfaces
  • Uizard: Generación de UI con IA
  • Galileo AI: Diseño de interfaces desde texto
