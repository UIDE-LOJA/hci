Esta actividad práctica te permitirá aplicar los principios de Gestalt en el diseño de interfaces. Te enfocarás en identificar problemas de diseño, aplicar principios de percepción visual, y mejorar interfaces usando herramientas de IA.

¿Qué realizarás?

Actividad Instrucciones específicas Entregable
Preparación Revisa la presentación de clase sobre principios de Gestalt: proximidad, similitud, continuidad, cierre, figura-fondo. Comprensión de principios de Gestalt.
Selección de interfaz Elige una interfaz existente (app, web, dashboard) que tenga problemas de organización visual o claridad. Captura de pantalla de interfaz "before".
Análisis de problemas Identifica qué principios de Gestalt están siendo violados o mal aplicados. Documenta los problemas específicos. Lista de problemas identificados.
Rediseño con IA Usa herramientas de IA (ChatGPT, Claude, Midjourney, Figma AI, etc.) para generar una versión mejorada aplicando principios de Gestalt. Captura de pantalla de interfaz "after".
Justificación Explica qué principios de Gestalt aplicaste y cómo mejoran la percepción visual y usabilidad de la interfaz. Justificación técnica de cambios.

Principios de Gestalt a considerar

  • Proximidad: Elementos cercanos se perciben como relacionados.
  • Similitud: Elementos similares se agrupan visualmente.
  • Continuidad: El ojo sigue líneas y patrones continuos.
  • Cierre: La mente completa formas incompletas.
  • Figura-fondo: Distinción clara entre elemento principal y fondo.
  • Simetría: Elementos simétricos se perciben como unidad.

Herramientas de IA sugeridas

  • ChatGPT/Claude: Para generar sugerencias de diseño y layouts.
  • Figma AI: Para rediseñar interfaces directamente.
  • Midjourney/DALL-E: Para generar mockups visuales.
  • Uizard: Para convertir sketches en diseños.
  • Galileo AI: Para generar interfaces desde descripciones.

Formato de entrega

  1. Interfaz "Before": Captura de pantalla original con anotaciones de problemas.
  2. Análisis: Lista de principios de Gestalt violados o mal aplicados.
  3. Interfaz "After": Captura de pantalla del rediseño con IA.
  4. Justificación: Explicación de qué principios aplicaste y cómo mejoran la interfaz.
  5. Herramienta usada: Menciona qué herramienta de IA utilizaste.

Materiales: Presentación de clase Semana 3, herramientas de IA, software de captura de pantalla.
